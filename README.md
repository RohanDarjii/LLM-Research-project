## 🔍 Project Overview

This project evaluates the performance of multiple Large Language Models (LLMs) — **ChatGPT, Gemini, and Groq** — on **discipline-specific exam questions** across:

- 💻 Computer Science  
- 📐 Mathematics  
- 🧠 Psychology  
- 📊 Economics  

### 🎯 Objectives
- Compare model accuracy under **standardized exam conditions**
- Analyze performance across **difficulty levels**:
  - Easy (Memory-based)
  - Medium (Conceptual)
  - Hard (Analytical)
- Perform **Before vs After prompt refinement analysis**
- Identify **error patterns and reasoning limitations**
- Validate results using **statistical tests (ANOVA)**
